# AI 治理百科

> 從全球法規、國際標準、倫理安全、企業治理、智財到台灣在地——人工智慧治理（AI Governance）的全景中文百科。

這是一個純靜態、可瀏覽、可搜尋的繁體中文知識網站，把分散的 AI 治理資料整理成 13 個面向：全球法規、國際標準、倫理與安全、企業治理、智財治理、台灣治理、地緣政治、案例研究、互動時間軸、可搜尋術語表、自我測驗與資料來源清單。內容以一篇智慧財產權治理的期刊專文為起點，再經多方查證、彙整自各國官方文件與權威來源（資料截至 2026 年 6 月）。

## ✨ 功能特色

- 📚 **13 個治理面向** — 法規、標準、倫理、企業、智財、台灣、地緣政治等各自獨立成頁，各有專屬網址與 SEO
- 🔍 **可搜尋術語表** — 上百個 AI 治理關鍵術語，可全文搜尋、可依七大類別篩選，中英對照
- 🕓 **互動時間軸** — 2016–2027 年 70+ 個重要里程碑，附來源連結
- 🧩 **自我測驗** — 隨堂測驗（即時對錯與解析、即時計分）＋ 可翻面字卡複習
- ⚖️ **案例研究** — 紐約時報 v. OpenAI、Getty v. Stability、NVIDIA–ARM、DeepSeek 等重大事件，含治理啟示
- 🔗 **深連結** — 卡片型頁面每個項目都有專屬 `#<slug>`，可直接分享
- 🌗 **深色 / 淺色模式** — 一鍵切換，並記憶偏好
- 📱 **響應式設計** — 手機、平板、桌機皆適配
- ⚡ **純靜態** — 無後端、載入快、可離線瀏覽

## 🛠 本機使用

```bash
# 1. clone 專案
git clone git@github.com:tingwei161803/ai-governance.git
cd ai-governance

# 2a. 最簡單：直接開啟首頁
open index.html

# 2b. 或啟動本機伺服器（建議，深連結 / 跨頁導覽才正常）
uv run python -m http.server 4173
# 然後瀏覽 http://localhost:4173
```

> 本專案為純靜態網站，不需安裝任何依賴。若要重建資料層或頁面，依序執行：
> `uv run python data/parse_research.py` →（子代理產生 `data/chunks/*.json`）→ `uv run python data/_assemble.py` → `uv run python data/gen_html.py`。所有 Python 一律用 `uv`。
